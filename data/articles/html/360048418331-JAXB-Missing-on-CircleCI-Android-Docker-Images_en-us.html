Removal of Java EE Modules

On August 17, 2020, the CircleCI Android Docker convenience images were upgraded from Java 8 to Java 11. You can read more about that announcement here: Android Convenience Image Moving to Java v11 on August 17th.

Since Java 11 no longer includes JAXB, this causes errors in builds. The following article can help describe how to enable JAXB for projects now that the Android images are running on Java 11: JAXB on Java 9, 10, 11 and beyond.
