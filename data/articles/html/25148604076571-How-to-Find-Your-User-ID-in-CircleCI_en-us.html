Overview

Your User ID in CircleCI is a unique identifier for your user account, often used for troubleshooting when working with CircleCI support, or identifying an actor when using the API.

This guide provides step-by-step instructions on how to locate your User ID within the CircleCI platform.

 

Locating Your User ID in CircleCI

  1. Log into CircleCI
  2. Go to the Home page (https://app.circleci.com/home/)
  3. Navigate to User Settings using the profile avatar
    Click on your profile picture located at the upper right hand of the Home page
  4. In the User Settings Page, it will display the Account Integrations Tab by default which contains a field with the User ID
