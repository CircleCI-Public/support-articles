Access Server Adit Logs via the UI
CircleCI Server customers can get Audit logs via the Admin Settings page in the UI or via the API.

 

Admin Settings

Admin users can visit the Admin Settings page while logged into CircleCI Server, and click "Audit Log". There you will see the option to download the full audit logs.

 

Troubleshooting

If you are experiencing issues generating Audit Logs via the UI, you can generate Audit Logs via the API.
