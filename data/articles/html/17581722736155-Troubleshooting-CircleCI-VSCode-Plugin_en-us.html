VSCode Plugin

When attempting to authenticate/login within VSCode, you may see that it times out with an error. If restarting VSCode, etc. does not resolve the issue, perform the following troubleshooting steps.

Troubleshooting Steps

Which Version of VSCode is being used on your machine?

  • The extension requires version 1.67 or higher

When was the token being used generated? We did revoke all project and personal API tokens before 00:00 UTC on January 5, 2023 in response to the 4 Jan Incident

  • Please try generating a new token on your user settings page and use that newly created token

If restarting VSCode did not help

  • Try removing the plugin/extension, closing VSCode completely, and reinstalling the CircleCI extension to see if that yields any changes in behavior

Next Steps

If you're still having trouble with the CircleCI VSCode Plugin, please reach out to our Support Team via the Support Portal.
