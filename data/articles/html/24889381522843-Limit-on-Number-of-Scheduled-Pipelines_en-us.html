Scheduled Pipelines

There is a limit of 50 scheduled pipelines per project. This limit cannot be increased at this time. 

Considerations

  • A scheduled pipeline can only be configured for one branch. If you need to schedule for two branches, you would need to set up two schedules.
  • Scheduled pipelines are not currently available for GitLab and GitHub App projects. To find out if you authorized your GitHub account through the GitHub OAuth app, or the GitHub App, see the GitHub App integration page.
