ERROR: Repository not found. fatal: Could not read from remote repository. Please make sure you have the correct access rights and the repository exists. Exited with code 128

If you receive this error, more than likely you are attempting to access a private repository from within your container without authentication. You will need to create a "User Key" SSH key and add it to your job environment so that it can access additional resources that require additional authentication.

More information on creating a user key can be found in our documentation.
