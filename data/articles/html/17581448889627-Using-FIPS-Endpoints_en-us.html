FIPS Endpoints with Terraform

With Terraform's official AWS provider, you can define the FIPS endpoint for RDS within the provider "aws" { ... } block. Specifically, you can set the use_fips_endpoint attribute to true, and then specify this endpoint value for RDS via the entrypoints attribute.
