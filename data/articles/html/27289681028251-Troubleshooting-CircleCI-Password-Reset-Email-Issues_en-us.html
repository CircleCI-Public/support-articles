Overview

Users may encounter issues when attempting to reset their CircleCI password, such as not receiving the password reset email. This can occur even when other notifications from CircleCI are being received. Understanding the nature of the account creation and the associated login method is crucial for resolving this issue.

Prerequisites

  • Access to the email account associated with your CircleCI account.
  • Knowledge of the account creation method (GitHub or email signup).

Instructions

If you're unable to receive a password reset email from CircleCI, follow these steps:

  1. Identify Account Creation Method: Determine if you created your CircleCI account using GitHub credentials.
  2. Login with GitHub: If you signed up with GitHub before September 2023, navigate to the CircleCI login page and select "Log In > Signed up with Github or Bitbucket before September 2023?".
  3. GitHub Password Reset: If you've forgotten your GitHub password, reset it directly through GitHub's password recovery process.
  4. Check Spam and Filters: If you signed up with an email, check your spam folder and email filters for messages from circleci.com.

Solution

For users who signed up with GitHub credentials, using the GitHub login option on CircleCI's login page is the correct method to access your account. The "Forgot Password?" feature will not work for these accounts. For email signups, ensure that emails from CircleCI are not being filtered or marked as spam.
