CircleCI Associate Developer

The purpose of CircleCI Associate Developer is to learn how to build, test, and deploy using CircleCI, from CI/CD basics to optimizations. You will then test your knowledge to earn a CircleCI certificate.

Intended for Beginners

This course will cover everything you need to get started using CircleCI. You will learn how to set up a CircleCI pipeline to build, test, and deploy an application. You will also learn some basic ways to simplify and optimize your configuration file.

This training course provides examples of topics outlined in CircleCI documentation.
