Learn concepts regarding migration from various CI/CD solutions to CircleCI.

In this course, you will learn concepts regarding migration from various CI/CD solutions to CircleCI. These trainings assume an understanding of how to clone repositories from version control systems (VCS) such as GitHub.

We will be adding lessons over time to this course, featuring both our old and new user interfaces (UIs).

 

Jenkins → CircleCI

Course SCORM: https://drive.google.com/file/d/1dVd5t-kRyxkUg-xVj13fneOCGeRKrghm/view?usp=share_link

By the end of this training, you will have an understanding of:

  • The differences between a Jenkins file and CircleCI config.yml file
  • How to begin migrating your projects
  • How to learn the configuration for CircleCI

This course is taught by Ryan Arlan, a CircleCI Senior DevOps Engineer.
