Overview

In some cases, a user may have signed up for CircleCI using their email + password. In this case you have signed up using the GitHub App integration or GitLab. 

 

How to connect this account using GitHub OAuth integration or BitBucket: 

To setup the integration, please navigate to your user settings page and click the "Connect" button.  https://app.circleci.com/settings/user
