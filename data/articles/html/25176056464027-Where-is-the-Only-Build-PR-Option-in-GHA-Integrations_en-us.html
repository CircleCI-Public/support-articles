Overview

Some users with GitHub Apps integrations may have noticed that there is no option to "Only Build Pull Requests" for GHA projects. This is unfortunately due to a limitation with GHA.

 

What's The Issue?

Due to the fact that GHA has a different back-end service compared to the GH Auth integration, the "Only Build Pull Requests" feature is not possible at this time. This is mentioned here in our documentation.
